# Guardian Eye V1

GitHub-ready package for the Guardian Eye H5 experience.

## Structure

- `index.html` — main V1 master file
- `data/` — reference JSON content files used to maintain the content system
- `docs/` — release notes and launch content pack

## Recommended GitHub setup

1. Create a new GitHub repository.
2. Upload everything in this folder to the repository root.
3. Keep the main file name as `index.html`.
4. If you want to use GitHub Pages, enable Pages and deploy from the repository root / main branch.

## Notes

- The current `index.html` already contains the content embedded, so it can run on its own.
- The JSON files in `data/` are kept for backup, future editing, and content expansion.
- Future edits should be made by copying this version first, rather than editing older branch files.
